# راهنمای رفع مشکل نمایش تصاویر در Localhost

## مشکلات احتمالی و راه‌حل‌ها

### 1. پاک کردن Cache Next.js

تصاویر ممکن است در cache ذخیره شده باشند. برای پاک کردن:

```bash
# پاک کردن پوشه .next
rm -rf .next

# یا در Windows PowerShell:
Remove-Item -Recurse -Force .next

# سپس سرور را دوباره راه‌اندازی کنید
npm run dev
```

### 2. بررسی اتصال اینترنت

تصاویر Unsplash نیاز به اتصال اینترنت دارند. مطمئن شوید:
- اینترنت شما متصل است
- فایروال یا VPN مانع دسترسی به Unsplash نشده باشد

### 3. بررسی Console مرورگر

در مرورگر (F12) به Console بروید و خطاها را بررسی کنید:
- خطاهای CORS
- خطاهای 404 (تصویر پیدا نشد)
- خطاهای شبکه

### 4. بررسی Network Tab

در Developer Tools:
1. به تب Network بروید
2. صفحه را Refresh کنید
3. فیلتر را روی "Img" تنظیم کنید
4. ببینید کدام تصاویر لود نمی‌شوند
5. روی تصویر کلیک کنید و Status Code را ببینید

### 5. تست مستقیم URL تصاویر

در مرورگر، یکی از URL های تصویر را مستقیماً باز کنید:
```
https://images.unsplash.com/photo-1628088062854-d1870b4553da?w=800&h=800&fit=crop&q=80
```

اگر تصویر نمایش داده نشد، مشکل از Unsplash است.

### 6. بررسی next.config.ts

مطمئن شوید که `images.unsplash.com` در `remotePatterns` است (که الان هست).

### 7. راه‌اندازی مجدد سرور

```bash
# متوقف کردن سرور (Ctrl+C)
# سپس دوباره راه‌اندازی:
npm run dev
```

### 8. استفاده از unoptimized در Development

اگر مشکل ادامه داشت، می‌توانید در `next.config.ts` این را اضافه کنید:

```typescript
unoptimized: process.env.NODE_ENV === "development" ? true : false,
```

### 9. بررسی Console در Terminal

خطاهای مربوط به تصاویر را در terminal که `npm run dev` را اجرا کرده‌اید بررسی کنید.

## راه‌حل سریع

1. **پاک کردن Cache:**
   ```bash
   Remove-Item -Recurse -Force .next
   npm run dev
   ```

2. **Hard Refresh در مرورگر:**
   - Windows: `Ctrl + Shift + R`
   - Mac: `Cmd + Shift + R`

3. **بررسی Console مرورگر:**
   - F12 → Console → بررسی خطاها

## اگر مشکل ادامه داشت

لطفاً این اطلاعات را بفرستید:
1. خطاهای Console مرورگر
2. Status Code تصاویر در Network Tab
3. آیا URL تصاویر در مرورگر باز می‌شوند؟
4. آیا اینترنت شما متصل است؟

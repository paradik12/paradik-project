# راهنمای بارگذاری تصاویر محصولات از Alibaba.com

## روش دریافت لینک مستقیم تصاویر

### مرحله 1: پیدا کردن تصویر محصول در Alibaba.com

1. به سایت [Alibaba.com](https://www.alibaba.com/) بروید
2. محصول مورد نظر را جستجو کنید
3. روی محصول کلیک کنید تا صفحه جزئیات باز شود

### مرحله 2: کپی کردن لینک تصویر

**روش A: از طریق مرورگر**
- راست کلیک روی تصویر محصول
- انتخاب "Copy image address" یا "Copy image URL"
- لینک کپی می‌شود

**روش B: از طریق DevTools**
1. F12 را فشار دهید (باز کردن Developer Tools)
2. روی تصویر راست کلیک کنید
3. "Inspect" را انتخاب کنید
4. در کد HTML، `src` تصویر را پیدا کنید
5. لینک را کپی کنید

### مرحله 3: فرمت لینک‌های Alibaba

لینک‌های تصاویر Alibaba معمولاً به این صورت هستند:
```
https://sc04.alicdn.com/kf/xxxxx/product-name.jpg
https://ae01.alicdn.com/kf/xxxxx/product-name.png
```

## نحوه ارسال لینک‌ها

لطفاً لینک‌ها را به این صورت ارسال کنید:

```
پسته درجه یک: https://sc04.alicdn.com/kf/xxx/pistachio.jpg
زعفران اصل: https://sc04.alicdn.com/kf/xxx/saffron.jpg
...
```

یا می‌توانید لینک صفحه محصول در Alibaba را بدهید تا من تصویر را استخراج کنم.

## نکات مهم

1. **کیفیت تصویر**: سعی کنید از تصاویر با کیفیت بالا استفاده کنید
2. **سایز**: تصاویر باید حداقل 400x400 پیکسل باشند
3. **فرمت**: JPG یا PNG
4. **حقوق کپی**: اطمینان حاصل کنید که حق استفاده از تصاویر را دارید

## جایگزین: استفاده از تصاویر Unsplash با کیفیت

اگر نمی‌توانید از Alibaba تصویر بگیرید، می‌توانم از تصاویر Unsplash با کیفیت بالا استفاده کنم که مربوط به همان محصولات هستند.
